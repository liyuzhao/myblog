文本经常需要和键盘打交道，现在记录一下初步的键盘相关的知识点。
键盘出现和消失促发两个消息。

UIKeyboardWillShowNotification （出现通知）
UIKeyboardWillHideNotification （隐藏通知）

NSNotificationCenter 监听两个消息即可捕捉到键盘弹出和隐藏的两个消息。
