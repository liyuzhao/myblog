发表于 2016年03月03日 | 分类于 java | | 阅读次数:

正则表达式匹配电话号码和手机号码

正则表达式（电话号码）一、
\d{3}-\d{8}|\d{4}-\d{7}
匹配 0511-1234567 | 022 -28191732
不匹配 02-123456 12345-123456
正则表达式（电话号码）二、
(^\([0]\d{2}|\d{4}\))(\d{6,7}$)
匹配    (021)1234567 | (0411)123456 | (000)000000
不匹配    (123)1234567 | 025123456 | 0252345678
